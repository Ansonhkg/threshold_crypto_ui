BLS Threshold Crypto

Interface for threshold_crypto. The rust library is compiled to web assembly and can be interacted with via the forms on this page.

WARNING: PROTOTYPE

This is a prototype tool and will definitely change in incompatible ways in the future. Do not ever use this tool for anything.

Derive Public Key From Secret Key

Input: Secret Key Hex (as per bincode serialization), 32 bytes ie 64 hex chars

Output: Public Key Hex, 48 bytes ie 96 hex chars

Sign Message With Secret Key

Input: Secret Key Hex (as per bincode serialization), 32 bytes ie 64 hex chars

Input: Message can be any ascii input. In the future hex encoded binary will be possible too.

Output: The signature for this message will appear here.

Verify Message With Public Key

Input: Public Key Hex, 48 bytes ie 96 hex chars

Input: Message can be any ascii input. In the future hex encoded binary will be possible too.

Output: Whether the signature is valid or not.
